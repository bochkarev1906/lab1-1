Car Rental System
=================

This project was created to manage a car rental system.

There are 7 classes in my project.

1) **Main.java**: you can make a request to create class objects (Booking, Car, Customer, CarCatalog) and display information about these objects.

2) **Booking.java**: there are the constructor of Booking objects and some functions.

3) **Car.java**: there are the constructor of Car objects and some functions.

4) **Customer.java**: there are the constructor of Customer objects and some functions.

5) **CarCatalog.java**: there are the constructor of CarCatalog objects and some functions.

6) **CarCatalogTest.java**: it's the testing class of CarCatalog class.

7) **CustomerTest.java**: it's the testing class of Customer class.
